# URL Shortener using Flask

## Description
This is a simple URL Shortener web application built using Python and Flask. It converts long URLs into shorter links and redirects users to the original URL.

## Features
- Shortens long URLs
- Generates unique short codes
- Redirects to original URL
- Uses SQLite database for storage

## Technologies Used
- Python
- Flask
- SQLite
